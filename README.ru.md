# Image Magnifier

> Userscript для Tampermonkey, который открывает любое изображение в плавающем окне с зумом по двойному нажатию Ctrl.  
> Панорамирование, масштабирование, изменение размера окна и аннотирование изображений инструментами рисования — карандаш, стрелка, линия и прямоугольник.

## Установка

### Шаг 1 — Установите менеджер userscript'ов

Для работы скрипта требуется браузерное расширение. Поддерживаемые менеджеры:

| Расширение | Chrome | Firefox | Edge |
|---|---|---|---|
| **Tampermonkey** (рекомендуется) | ✓ | ✓ | ✓ |
| Violentmonkey | ✓ | ✓ | ✓ |
| Greasemonkey | — | ✓ | — |

Установите расширение из официального магазина дополнений вашего браузера.

### Шаг 2 — Установите скрипт

**Вариант А — через Raw-ссылку на GitHub (проще всего):**

1. Откройте `imgMagnifier.js` на GitHub
2. Нажмите кнопку **Raw**
3. Tampermonkey автоматически определит скрипт и предложит установить его
4. Нажмите **Установить**

**Вариант Б — вручную:**

1. Скопируйте содержимое файла `imgMagnifier.js`
2. Откройте Tampermonkey → **Создать новый скрипт**
3. Вставьте код и нажмите **Сохранить** (`Ctrl+S`)

### Шаг 3 — Используйте на любой странице

Откройте любой сайт с изображениями. Наведите курсор на картинку и нажмите **Ctrl дважды** — откроется окно лупы.

## Как пользоваться

1. Наведите курсор мыши на любое изображение на веб-странице
2. Быстро нажмите **Ctrl дважды** — изображение откроется в плавающем окне
3. **Прокручивайте** колёсико мыши для масштабирования (зум привязан к позиции курсора)
4. **Нажмите и тащите** внутри области просмотра для панорамирования
5. **Тащите** за заголовок окна, чтобы переместить его
6. **Изменяйте размер** окна за правый нижний угол
7. Нажмите **Ctrl дважды**, **Esc** или кликните за пределами окна, чтобы закрыть его

## Возможности

### Масштабирование

Зум колёсиком мыши привязан к позиции курсора — точка под курсором остаётся на месте, а всё остальное масштабируется вокруг неё. Диапазон масштаба: от 5% до 2000%.

Текущий уровень масштаба отображается в заголовке окна.

### Кнопки панели управления

| Кнопка | Действие |
|---|---|
| **−** | Отдалить (по центру) |
| **+** | Приблизить (по центру) |
| **⊡** | Вписать изображение в окно |
| **×** | Закрыть окно |

### Инструменты рисования

Панель под заголовком предоставляет инструменты аннотирования:

| Инструмент | Описание |
|---|---|
| **Перемещение** | Панорамирование изображения (режим по умолчанию) |
| **Карандаш** | Произвольное рисование от руки |
| **Стрелка** | Нарисовать стрелку между двумя точками |
| **Линия** | Нарисовать прямую линию |
| **Прямоугольник** | Нарисовать прямоугольник |


### Закрытие окна

Окно можно закрыть:
- Нажатием **Esc**
- Повторным двойным нажатием **Ctrl**
- Кликом в любом месте за пределами окна

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Ctrl` × 2 | Открыть / закрыть лупу |
| `Ctrl+Z` | Отменить последнюю аннотацию |
| `Esc` | Закрыть окно |
